CCPi Core Imaging Library (CIL)
===============================
CCPi Core Imaging Library (CIL) is a library of algorithms that are used in Computed Tomography(CT). CCPi project is funded by EPSRC UK for supporting the CT community.
This library provides a modules for several steps involved in the CT. The algorithms are contributed by the community and the code is re-engineered by the CCPi funded staff
to make them run faster and maintainable.
